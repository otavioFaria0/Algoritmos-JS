# Algoritmos-JS (Monitoria)

**Práticas de Algoritmos e Lógica de Programação**, organizado para facilitar leitura e avaliação.

## Conteúdos

### Ordenação (Sorting)
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### Busca (Search)
- Busca Binária (Binary Search)

## Como executar (Node.js)

Requisitos: **Node.js 18+**.

Instale dependências (não há libs externas, mas o `npm` facilita os scripts):

```bash
npm install
```

### Rodar exemplos

```bash
npm run selection
npm run insertion
npm run merge
npm run quick
npm run binary
```

## Estrutura

- `src/` → implementações dos algoritmos
- `src/data/` → dados de exemplo (lista de livros)
- `examples/` → scripts simples para executar e ver o resultado
